# Georgian Month Names Reference

## All 12 Months in Georgian

| Month # | Georgian Name | English Name |
|---------|---------------|--------------|
| 1 | იანვარი | January |
| 2 | თებერვალი | February |
| 3 | მარტი | March |
| 4 | აპრილი | April |
| 5 | მაისი | May |
| 6 | ივნისი | June |
| 7 | ივლისი | July |
| 8 | აგვისტო | August |
| 9 | სექტემბერი | September |
| 10 | ოქტომბერი | October |
| 11 | ნოემბერი | November |
| 12 | დეკემბერი | December |

## Usage Examples

### Full Date Format (Default)
```jinja
{{ date2(docs.date_field) }}
```
**Output:** `16 იანვარი 2025`

### Short Format (Without Year)
```jinja
{{ date2(docs.date_field, 'short') }}
```
**Output:** `16 იანვარი`

### Month and Year Only
```jinja
{{ date2(docs.date_field, 'month_year') }}
```
**Output:** `იანვარი 2025`

## Common Use Cases

### Contract Date (ხელშეკრულების თარიღი)
```jinja
ხელშეკრულება დადებულია {{ date2(docs.date_contract) }} წელს
```
**Output:** `ხელშეკრულება დადებულია 16 იანვარი 2025 წელს`

### Birth Date (დაბადების თარიღი)
```jinja
დაბადების თარიღი: {{ date2(docs.birthday) }}
```
**Output:** `დაბადების თარიღი: 5 მარტი 1990`

### Employment Date (დასაქმების თარიღი)
```jinja
სამსახურში მიღების თარიღი: {{ date2(docs.date_join) }}
```
**Output:** `სამსახურში მიღების თარიღი: 1 აპრილი 2020`

### Invoice Date (ინვოისის თარიღი)
```jinja
გადახდის თარიღი: {{ date2(docs.invoice_date) }}
ვადა: {{ date2(docs.invoice_date_due) }}
```
**Output:** 
```
გადახდის თარიღი: 15 მაისი 2025
ვადა: 30 მაისი 2025
```

### Period Range (პერიოდი)
```jinja
პერიოდი: {{ date2(docs.date_start, 'short') }} - {{ date2(docs.date_end, 'short') }}
```
**Output:** `პერიოდი: 1 იანვარი - 31 დეკემბერი`

### Month Report (თვიური ანგარიში)
```jinja
ანგარიში {{ date2(docs.report_date, 'month_year') }}-ის თვისთვის
```
**Output:** `ანგარიში იანვარი 2025-ის თვისთვის`

## date1 — Simple dd/mm/yyyy Format

### Usage
```jinja
{{ date1(docs.date_field) }}
```

### All Months Example
```
date1('2026-01-05') → 05/01/2026
date1('2026-02-05') → 05/02/2026
date1('2026-03-05') → 05/03/2026
date1('2026-04-05') → 05/04/2026
date1('2026-05-05') → 05/05/2026
date1('2026-06-05') → 05/06/2026
date1('2026-07-05') → 05/07/2026
date1('2026-08-05') → 05/08/2026
date1('2026-09-05') → 05/09/2026
date1('2026-10-05') → 05/10/2026
date1('2026-11-05') → 05/11/2026
date1('2026-12-05') → 05/12/2026
```

### Common Use Cases
```jinja
{{ date1(docs.invoice_date) }}
```
**Output:** `05/01/2026`

```jinja
{{ date1(docs.date_start) }} - {{ date1(docs.date_end) }}
```
**Output:** `01/01/2026 - 31/12/2026`

## date3 — Georgian Formal Date Format (YYYY წლის DD Month)

### Signature
```jinja
{{ date3(docs.date_field) }}
{{ date3(docs.date_field, 'დან') }}
{{ date3(docs.date_field, 'მდე') }}
{{ date3(docs.date_field, 'ის') }}
```

### All Months — No Suffix (default)
```
date3('2026-01-05') → 2026 წლის 05 იანვარი
date3('2026-02-05') → 2026 წლის 05 თებერვალი
date3('2026-03-05') → 2026 წლის 05 მარტი
date3('2026-04-05') → 2026 წლის 05 აპრილი
date3('2026-05-05') → 2026 წლის 05 მაისი
date3('2026-06-05') → 2026 წლის 05 ივნისი
date3('2026-07-05') → 2026 წლის 05 ივლისი
date3('2026-08-05') → 2026 წლის 05 აგვისტო
date3('2026-09-05') → 2026 წლის 05 სექტემბერი
date3('2026-10-05') → 2026 წლის 05 ოქტომბერი
date3('2026-11-05') → 2026 წლის 05 ნოემბერი
date3('2026-12-05') → 2026 წლის 05 დეკემბერი
```

### All Months — Suffix `'დან'` (from)
```
date3('2026-01-05', 'დან') → 2026 წლის 05 იანვრიდან
date3('2026-02-05', 'დან') → 2026 წლის 05 თებერვლიდან
date3('2026-03-05', 'დან') → 2026 წლის 05 მარტიდან
date3('2026-04-05', 'დან') → 2026 წლის 05 აპრილიდან
date3('2026-05-05', 'დან') → 2026 წლის 05 მაისიდან
date3('2026-06-05', 'დან') → 2026 წლის 05 ივნისიდან
date3('2026-07-05', 'დან') → 2026 წლის 05 ივლისიდან
date3('2026-08-05', 'დან') → 2026 წლის 05 აგვისტოდან
date3('2026-09-05', 'დან') → 2026 წლის 05 სექტემბრიდან
date3('2026-10-05', 'დან') → 2026 წლის 05 ოქტომბრიდან
date3('2026-11-05', 'დან') → 2026 წლის 05 ნოემბრიდან
date3('2026-12-05', 'დან') → 2026 წლის 05 დეკემბრიდან
```

### All Months — Suffix `'მდე'` (until)
```
date3('2026-01-05', 'მდე') → 2026 წლის 05 იანვრამდე
date3('2026-02-05', 'მდე') → 2026 წლის 05 თებერვლამდე
date3('2026-03-05', 'მდე') → 2026 წლის 05 მარტამდე
date3('2026-04-05', 'მდე') → 2026 წლის 05 აპრილამდე
date3('2026-05-05', 'მდე') → 2026 წლის 05 მაისამდე
date3('2026-06-05', 'მდე') → 2026 წლის 05 ივნისამდე
date3('2026-07-05', 'მდე') → 2026 წლის 05 ივლისამდე
date3('2026-08-05', 'მდე') → 2026 წლის 05 აგვისტომდე
date3('2026-09-05', 'მდე') → 2026 წლის 05 სექტემბრამდე
date3('2026-10-05', 'მდე') → 2026 წლის 05 ოქტომბრამდე
date3('2026-11-05', 'მდე') → 2026 წლის 05 ნოემბრამდე
date3('2026-12-05', 'მდე') → 2026 წლის 05 დეკემბრამდე
```

### All Months — Suffix `'ის'` (genitive)
```
date3('2026-01-05', 'ის') → 2026 წლის 05, იანვრის
date3('2026-02-05', 'ის') → 2026 წლის 05, თებერვლის
date3('2026-03-05', 'ის') → 2026 წლის 05, მარტის
date3('2026-04-05', 'ის') → 2026 წლის 05, აპრილის
date3('2026-05-05', 'ის') → 2026 წლის 05, მაისის
date3('2026-06-05', 'ის') → 2026 წლის 05, ივნისის
date3('2026-07-05', 'ის') → 2026 წლის 05, ივლისის
date3('2026-08-05', 'ის') → 2026 წლის 05, აგვისტოს
date3('2026-09-05', 'ის') → 2026 წლის 05, სექტემბრის
date3('2026-10-05', 'ის') → 2026 წლის 05, ოქტომბრის
date3('2026-11-05', 'ის') → 2026 წლის 05, ნოემბრის
date3('2026-12-05', 'ის') → 2026 წლის 05, დეკემბრის
```

### Common Use Cases
```jinja
ხელშეკრულება გაფორმდა {{ date3(docs.date_start) }}
```
**Output:** `ხელშეკრულება გაფორმდა 2026 წლის 05 იანვარი`

```jinja
მოქმედია {{ date3(docs.date_start, 'დან') }} {{ date3(docs.date_end, 'მდე') }}
```
**Output:** `მოქმედია 2026 წლის 05 იანვრიდან 2026 წლის 31 დეკემბრამდე`

```jinja
{{ date3(docs.date_field, 'ის') }} მდგომარეობით
```
**Output:** `2026 წლის 05, იანვრის მდგომარეობით`
