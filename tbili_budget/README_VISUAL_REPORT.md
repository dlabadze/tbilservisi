# ბიუჯეტის ვიზუალური ანგარიში (Budget Visual Report)

## აღწერა

ეს მოდული ამატებს ვიზუალურ ანგარიშს ბიუჯეტის მოდულში, რომელიც საშუალებას იძლევა იხილოთ და გააანალიზოთ შესყიდვების გეგმებისა და ბიუჯეტ CPV-ების მონაცემები.

## ფუნქციონალი

### ძირითადი ფუნქციები:
- 📊 **ვიზუალური ანგარიშები**: წრიული და სვეტოვანი დიაგრამები
- 📋 **დეტალური ცხრილები**: ყველა მონაცემის დეტალური ჩვენება
- 📈 **შეჯამება**: ძირითადი მეტრიკების კარდები
- 📤 **ექსპორტი**: CSV ფორმატში მონაცემების გატანა
- 🔍 **ფილტრაცია**: შესყიდვების გეგმისა და ბიუჯეტ CPV-ის მიხედვით

### მონაცემები რომლებიც ჩანს:
- `amount` - გამოყენებული თანხა
- `budget_amount` - ბიუჯეტის თანხა
- `pu_re_am` - დარჩენილი რესურსი
- `selected_plan_name` - გეგმის დასახელება
- `cpv_id` - CPV კოდი

## როგორ გამოვიყენოთ

### 1. მენიუში გადასვლა
```
Accounting → Purchase Plans → ვიზუალური ანგარიში
```

### 2. ფილტრების არჩევა
1. **შესყიდვების გეგმა**: აირჩიეთ სასურველი შესყიდვების გეგმა
2. **ბიუჯეტ CPV**: აირჩიეთ კონკრეტული CPV კოდი
3. დააჭირეთ **"ანგარიშის გენერირება"** ღილაკს

### 3. ანგარიშის ნახვა
ანგარიში შეიცავს:
- **შეჯამების კარდები**: ძირითადი მეტრიკები
- **წრიული დიაგრამა**: ბიუჯეტის თანხების განაწილება
- **სვეტოვანი დიაგრამა**: თანხების შედარება
- **დეტალური ცხრილი**: ყველა ხაზის ინფორმაცია

### 4. ექსპორტი
დააჭირეთ **"ექსპორტი CSV"** ღილაკს მონაცემების ჩამოსატვირთად.

## ტექნიკური დეტალები

### ფაილების სტრუქტურა:
```
budget/
├── controllers/
│   ├── __init__.py
│   └── budget_report.py          # კონტროლერები
├── views/
│   └── visual_report_template.xml # HTML შაბლონი
├── static/src/js/
│   └── main.js                    # JavaScript ფუნქციები
└── README_VISUAL_REPORT.md        # ეს ფაილი
```

### API Endpoints:
- `/budget/visual_report` - მთავარი გვერდი
- `/budget/get_budget_cpvs` - CPV-ების მიღება
- `/budget/get_cpv_lines_data` - დეტალური მონაცემები
- `/budget/get_chart_data` - დიაგრამების მონაცემები

### მოდელების კავშირები:
```
purchase.plan → purchase.plan.line → budget.cpv → budget.cpv.line → budget.line
```

## შეცდომების მოგვარება

### ყველაზე ხშირი პრობლემები:
